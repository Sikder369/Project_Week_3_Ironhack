# Airbnb in New York City Analysis

This project explores the Airbnb market in New York City by analyzing how location, crime, reviews, and host behavior influence listing prices.

## Project Objective

The aim of this analysis is to test the following questions and hypotheses:

* Airbnb room type composition per district
* Relationship between price and crime level
* Hypothesis: higher crime, lower prices?
* Correlation between reviews and price
* Price comparison between multiple-listing hosts and single-listing hosts

## Dataset and Variables

The project uses Airbnb listing data from New York City together with crime data collected through an API.

Main variables analyzed include:

* Borough / district
* Room type
* Price
* Number of reviews
* Crime level
* Host listing type (multiple vs single listing)

## Key Findings

* Manhattan has the highest concentration of expensive entire-home listings.
* Brooklyn and Queens offer more affordable and diverse room options.
* Crime has a partial negative relationship with Airbnb prices.
* The correlation between reviews and price is very weak (`-0.0729`).
* Hosts with multiple listings have lower average prices than single-listing hosts.

## Conclusion

Airbnb prices in New York City are influenced by several factors, especially borough, room type, crime level, and host behavior.
The results show that pricing is not determined by one single variable, but by a combination of market and location-based factors.

## Recommendations

### For Hosts

* Use dynamic pricing based on demand, location, and safety trends.
* Improve listing quality and encourage reviews to attract more bookings.

### For Policymakers

* Monitor multi-listing hosts to maintain fair market competition.
* Promote transparent neighborhood safety data for better decision-making.
